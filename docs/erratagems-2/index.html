Erratagems - Driven by Irritation - Set Two

If, like me, you are sometimes irritated by WotC's choices on some of the cards, you may want to use these ZERO-STAR stratagems to fix these GLARING errors. These have been solidly tested, in that their existence lessens my irritation. Your mileage may vary.

NOTE: These aren't related to any other group (Aequitas, Alpha Trion, Ark, Bayformers, Reformatted, Edge of Iacon, Legendary Void, or anyone else); this is just a tongue-in-cheek set of images for a smile, that might be something folk might play with if they ever wanted to ignore an irritating lack of a trait.

This is the second set in the Erratagem cycle. The first set is available here.
